How-to Guide

Step 1: Creating a New Canvas

To create a new canvas, simply click the "New Canvas" button in the top left corner of the editor. You can then select the desired canvas size or start with a blank canvas.

Step 2: Using the Brush Tool

To use the brush tool, click the "Brush" icon in the toolbar. You can then select different brush styles, change the color by clicking the color swatch, and adjust the brush size using the slider.

Step 3: Adding Shapes and Text

To add shapes, click the corresponding icon in the toolbar (rectangle, circle, etc.). You can then customize the shape's size, color, and other properties. To add text, click the "Text" icon and then click on the canvas to start typing.

Step 4: Saving and Exporting

To save your canvas, click the "Save" button in the top right corner. You can choose to save the canvas as a PNG, JPEG, or SVG file. To export the canvas, click the "Export" button and select the desired file format.
